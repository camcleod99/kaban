# KabanDon

KanbanDon is a kanban board application that allows users to create, update, and delete tasks. 
Users can also move tasks between columns and mark tasks as complete.

## Technologies
- React
- Node.js
- Next.js
- Pocketbase
- Tailwind CSS
- TypeScript
- ESLint
